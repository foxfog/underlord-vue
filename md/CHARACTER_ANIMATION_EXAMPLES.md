# Примеры анимированного появления персонажей

## Базовый синтаксис

```json
{
  "type": "show",
  "character": "character-id",
  "position": {"l": 20},
  "from": {"l": -25},
  "duration": 2000
}
```

## Примеры использования

### Пример 1: Появление с анимацией слева (стандартно)

```json
{
  "type": "show",
  "character": "mc",
  "position": {"l": 20},
  "from": {"l": -25},
  "duration": 2000
}
```

**Результат**: Персонаж появится с левого края экрана (позиция -25%) и переместится на 20% за 2 секунды.

### Пример 2: Появление справа

```json
{
  "type": "show",
  "character": "npc",
  "position": {"r": 20},
  "from": {"r": -30},
  "duration": 1.5
}
```

**Результат**: Персонаж появится справа и переместится влево за 1.5 секунды.

### Пример 3: Без параметра duration (по умолчанию 1s)

```json
{
  "type": "show",
  "character": "character",
  "position": {"l": 50},
  "from": {"l": 0}
}
```

**Результат**: Когда `from` указан, но `duration` отсутствует, используется 1m.

### Пример 4: Без анимации (без from)

```json
{
  "type": "show",
  "character": "character",
  "position": {"l": 30}
}
```

**Результат**: Персонаж просто появляется в позиции 30% без анимации (если `from` не указан).

## Сложные сценарии

### Несколько персонажей с разными скоростями

```json
{
  "id": "multi_character_scene",
  "steps": [
    {
      "type": "scene",
      "id": "room"
    },
    {
      "type": "show",
      "character": "hero",
      "position": {"l": 25},
      "from": {"l": -40},
      "duration": 1200
    },
    {
      "type": "show",
      "character": "villain",
      "position": {"r": 25},
      "from": {"r": -40},
      "duration": 1800
    },
    {
      "type": "dialogue",
      "character": "hero",
      "text": "Давайте разберемся!"
    },
    {
      "type": "dialogue",
      "character": "villain",
      "text": "Если сможешь!"
    }
  ]
}
```

### Последовательное появление

```json
{
  "type": "show",
  "character": "person1",
  "position": {"l": 10},
  "from": {"l": -50},
  "duration": 1000
}
```

```json
{
  "type": "dialogue",
  "text": "Подождём второго..."
}
```

```json
{
  "type": "show",
  "character": "person2",
  "position": {"r": 10},
  "from": {"r": -50},
  "duration": 1500
}
```

## Доступные параметры позиции

- **l** - левая позиция (в процентах от левого края)
- **r** - правая позиция (в процентах от правого края)
- **t** - верхняя позиция (в процентах от верхнего края)
- **b** - нижняя позиция (в процентах от нижнего края)

## Тестирование

Используйте файл `animation_test.json` для проверки всех вариантов анимации.

## Особенности

✅ Плавная анимация с ease-in-out функцией плавности
✅ По умолчанию 1s, если duration не указана
✅ Одновременная анимация нескольких персонажей
✅ Работает с любыми позициями и размерами
✅ Поддерживает CSS классы через параметр `class`
