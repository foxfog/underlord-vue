# Система Инвентаря

## Структура предметов

Все предметы определяются в файлах:
- `/data/items/equipment.json` - экипировка (одежда, оружие, украшения)
- `/data/items/other.json` - расходники, валюта, материалы, еда

### Свойство `stackable`

Все предметы могут иметь свойство `stackable`:
- **`stackable` отсутствует или `false`** (по умолчанию) - предмет НЕ стакается, каждый экземпляр занимает отдельный слот (обычно это экипировка)
- **`stackable: true`** - предметы этого типа могут стакаться и образовывать группы (например, монеты, расходники)

## Структура инвентаря в values.json

Инвентарь персонажа хранится в поле `inventory`:

```json
{
  "id": "mc",
  "inventory": {
    "items": [
      {
        "itemId": "gasmask" // Для нестакаемых предметов quantity = 1 по умолчанию
      },
      {
        "itemId": "ygdrasil-coin-old",
        "quantity": 23      // Для стакаемых явно указывается количество
      }
    ]
  }
}
```

## Контекстное меню (ПКМ)

Добавлено контекстное меню (правый клик) для предметов в инвентаре и оборудовании.
**Файл:** `src/renderer/components/game/inventory/ContextMenu.vue`

### Функционал

**Предметы в инвентаре:**
При правом клике появляются опции:
- **Выбросить**
- **Одеть на [часть тела]**

**Одетые предметы:**
При правом клике на одетый предмет (на теле персонажа):
- **Выбросить**
- **Снять** (возвращает предмет в инвентарь)

### Интеграция
Взаимодействие идет через emit в родительский компонент `Game.vue`, где реализованы обработчики `handleEquip`, `handleUnequip` и `handleSwap`.

> [!TIP]
> Названия слотов локализуются автоматически ("head" → "голова", "chest" → "грудь").
